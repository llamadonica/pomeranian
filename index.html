    Pomeranian

    A simple, themeable Pomodoro Timer for Linux and eventually Windows.

    What you need

    Pomeranian requires

    • A recent version of Python.
    • Vala-0.16 or better.
    • GCC
    • Development packages for:
      • Glib (>= 2.32)
      • GTK+-3.0 (>= 3.4)
      • GStreamer (>= 0.10)
      • libgee
      • libcanberra and the GTK interoperability packages.

    Getting it

    On the command line type:

    git clone git://github.com/llamadonica/pomeranian.git

    Compiling

    On the command line type:

    ./configure
make
sudo make install
